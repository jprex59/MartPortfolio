Alchemy

What is Alchemy ?

Alchemy is an auto checkout extension that allows you to buy limited shoes on different European sites such as Zalando. The interface has been designed to be as simple as possible, the extension works on any devices that have Google Chrome, so it’s windows, mac, and linux compatible.

How i made Alchemy ?

I really think that Alchemy is the solution, because it doesn’t need to bypass any site security thanks to the browser things, as the extension is linked to google chrome, the cookies we need to access to the website are already generated. This is the simple way to get limited shoes.
